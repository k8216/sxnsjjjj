Ques.7:Define the following terms:
(a) Class


Ans.7(a)A class is a blueprint or template for creating objects in object-oriented programming (OOP). It defines a set of attributes (data) and methods (functions) that the created objects (instances) will have. Classes encapsulate data for the object and methods to manipulate that data.

(b) Object

Ans.7(b):An object is an instance of a class. It is created using the class definition and contains the data (attributes) and behavior (methods) defined in the class. Each object can have different values for its attributes.

(c) Polymorphism


Ans.7(c)Polymorphism is a concept in OOP that allows objects of different classes to be treated as objects of a common superclass. It enables methods to do different things based on the object calling them. This can be achieved through method overriding or operator overloading.

(d) Lamda function

Ans.7(d)
A lambda function is a small, anonymous function defined using the lambda keyword in Python. It can take any number of arguments but can only have one expression. Lambda functions are often used for short, throwaway functions where defining a full function would be unnecessary.

(e) Function overriding

Ans.7(e)
Function overriding occurs when a subclass provides a specific implementation of a method that is already defined in its superclass. The method in the subclass has the same name and parameters as the one in the superclass, allowing the subclass to provide its own behavior.

(f) Use of Super keyword

Ans.7(f):
The super keyword in Python is used to call methods from a parent class. It is particularly useful in method overriding and allows you to access inherited methods that have been overridden in a subclass.
